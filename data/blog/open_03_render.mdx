# 生产版本以开发版本
安装开发依赖，通过启动命令开发服务器
```
npm install
npm run dev
```

生产版本
部署到生产环境，我们建立在生产版本的应用程序
```
npm run build
```
build命令创建优化后的生产版本，代码最小化，删除不必要的调试信息

将应用程序部署到Web服务器
