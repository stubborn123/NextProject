
使用React Query 将数据同步至服务器

优化性能
React Query 是服务器状态的库，负责管理服务器和客户端之间的异步操作
Redux等是客户端状态的库，存储异步数据

因此，React Query 是一个在前端维护服务器状态的库，即作为服务器存储内容的缓存。React Query 简化了对服务器数据的处理，在某些情况下，可以消除将服务器数据存储在前端的需求。
